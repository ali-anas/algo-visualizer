
# Path Algorithm Visualizer

## Specs
- When an algorithm is running, the grid and play/clear controls are freezed and are not clickable until algorithm execution completes.

## Libraries Used
- Front end library - React
- State Management - [Redux Toolkit](https://redux-toolkit.js.org/)
- App Tour - [React Joyride](https://docs.react-joyride.com/)
- Toast Components - [Sooner](https://sonner.emilkowal.ski/)
- Icons - [Lucide React](https://lucide.dev/)

### Build Libraries
  - Module Bundler - [Vite](https://vitejs.dev/)
  - CSS Compiler - [Sass](https://sass-lang.com/)

## To run in local
  1. `git clone https://github.com/ali-anas/algo-visualizer.git`
  2. `cd algo-visualizer`
  3. `yarn`
  4. `yarn run dev`
  5. local version will be running on `localhost:5173` by default.
